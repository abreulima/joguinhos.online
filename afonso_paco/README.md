# Climb Higher - Web Edition

Open `index.html` in a browser to play the browser version of the game.

Controls:
- `A/D` or arrow keys to move
- `W` or `Space` to jump
- Left click or `J` to attack
- Right click to shield
- `V` to toggle god mode and flight
- `B` to jump straight to the boss
- `Enter` to start or skip the victory scene

Progress is saved in `localStorage` for the browser version.
