# Versao Web

Esta pasta contem uma versao do jogo feita para correr no navegador sem alterar o `jogo.py` original.

## Como abrir

1. Inicia um servidor local na raiz do projeto.
2. Abre `web/index.html` no navegador.

Exemplo com Python:

```bash
python -m http.server 8000
```

Depois abre `http://localhost:8000/web/`.
